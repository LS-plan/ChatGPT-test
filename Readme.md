**# github源**

```
git clone --depth=1 https://github.com/LS-plan/ChatGPT-test.git ./plugins/chatgpt-test/
```

```
git clone --depth=1 https://gitee.com/LS-plan/ChatGPT-test.git ./plugins/chatgpt-test/

```
cd plugins/chatgpt-test

npm i
